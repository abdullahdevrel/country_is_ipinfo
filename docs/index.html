Country

Country is a browser-friendly, privacy-conscious geolocation API that gets a user's country (and nothing else) from their IP address.

Usage

Country has a minimal :fire: interface.

Query yourself.

{% include window.html address="https://api.country.is" %}

Query any IP.

{% include window.html address="https://api.country.is/9.9.9.9" %}

Country also works with IPv6.

{% include window.html address="https://api.country.is/::ffff:909:909" %}

See the age of the underlying data file.

{% include window.html address="https://api.country.is/version" %}

Country automatically checks for new data every 24 hours.

Deployment

If you prefer not to use the hosted service, run your own private server with Docker.

Notes

Country uses GeoLite2 data created by MaxMind. Since 30 December 2019, you need to register for a license key to download their data.

The hosted service does not log requests.
